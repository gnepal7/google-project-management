# Common myths about project managers
  1. Myth: You have to be an expert in the field and 
    have a lot of technical knowledge about the project.
    => The bottom line :- Project manager don't need to be experts in every field.
  2. Myth: Only people with a lot of experience within an 
     organization can manage projects successfully. 
    => The bottom line :- It is essential to master the skills, tools, and techniques.
  3. Myth: You have to know every single detail about the project at all times. 
    => Through the communication skills and strategic approach for problem-solving
       you can provide solutions and help remove barriers for your team.
